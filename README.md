# Customer Systems Presentation

## Folder Structure

```
presentation/
├── slides/          # Presentation slides (HTML, PPTX, or other formats)
├── assets/          # Media and resource files
│   └── images/      # Images, diagrams, screenshots
├── notes/           # Speaker notes and preparation materials
├── scripts/         # Automation scripts and configurations

```

## Purpose

This folder contains all materials related to the Customer Systems presentation, including slides, supporting assets, speaker notes, and any automation scripts.

## Usage

- **slides/**: Place your presentation files here
- **assets/images/**: Add images, diagrams, or screenshots that support your presentation
- **notes/**: Store speaker notes, talking points, or Q&A preparation
- **scripts/**: Keep any automation scripts, data processing tools, or utilities here
